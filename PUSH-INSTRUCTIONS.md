## CÁCH PUSH (2 PHÚT)

### Bước 1: Mở GitHub Desktop
- Click icon GitHub Desktop
- Hoặc search "GitHub Desktop" trong Start Menu

### Bước 2: Commit
1. GitHub Desktop sẽ hiện changes
2. Nhập commit message:
   ```
   Add Office View - Personal MBA Layout
   ```
3. Click **"Commit to main"**

### Bước 3: Push
1. Click **"Push origin"**
2. Đợi thanh progress chạy xong (~30 giây)

### Bước 4: Xong!
Đợi 2-3 phút deploy, sau đó mở:
