## 🔄 WEEKLY REVIEW PROCESS

**When:** Sunday 20:00 Vietnam Time

**Steps:**
1. **Review** all files created this week
2. **Categorize** into 5 functions
3. **Archive** outdated information
4. **Update** MEMORY.md with key insights
5. **Plan** next week's priorities

## 🎯 DECISION FRAMEWORK

Before any significant action, verify:

1. **Value Creation** - Does this solve a real problem?
2. **Marketing** - Is this clear to user?
3. **Sales** - Is this easy to use?
4. **Value Delivery** - Can we deliver reliably?
5. **Finance** - Is this efficient?

**Red Flags (stop and reassess):**
- Building without validating need
- Complex solutions when simple works
- Making assumptions without checking
- Wasting resources on low-value tasks
